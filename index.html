Hi, welcome to JS simplified. The aim of this introductory crash course is to help trigger / develop 'application instinct'; right from the very first time you set your eyes on a concept in Native Javascript. A number of things are very important if you will be able to apply what you've learnt to projects at any level: thorough understanding of concepts, how they work, how they can be put to use and what they can be useful for. Creating your own projects become easier not only by studying materials, but also by observing how others have been able to use them in simple projects.

Content

Js simplified is a project turned study material. Content is for those either new to Native Js or have had prior contact with Native Js but having a hard time applying what they've learnt so far in projects. It focuses mainly on using javascript for front-end projects. You will learn to write one code block in many ways, building simple programs along the way. Please note that this is NOT (necessarily) an 'animation course', though some of the practical examples are materials used in building the JS simplified site (homepage) from scratch.
